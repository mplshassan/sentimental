# What is *sentimental*?
Sentimental uses a sentiment analyzer (VADER) to plot the compound score of multiple sentences.

## Features
- **free-text mode**: type in sentences one-by-one or paste huge chunks. You decide!
- **sentence-by-sentence mode**: smaller viewer made for intentional, incremental typing

## Setup
```bash
git clone https://github.com/mplshassan/sentimental
cd sentimental
pip install -r requirements.txt
streamlit run app.py
```
